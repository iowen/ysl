Image Autofit &
Subject Selection

Image Autofit is an awesome features that allows you to responsive your images automatically by cutting its edges and resizing its sizes, to preserve a high definition on every device. With Image Autofit you can also select the position of your main subject in the image to avoid to cut a person on a image or focus the viewer’s attention on a particular section of it.

Auto Check Errors

Infinity Slider was build with an advanced script that provide you an important feature, the AutoCheckErrors. This feature will looking for errors and help you to configure your slider, avoiding misspell of classes names, times, delays or effects.

Multi Instance Slider

Infinity Slider is ready to be integrated infinity times. you can create so many slider as you wish, in the same page. Are you surprise? You can even create a slider inside the slider, this is what multi instance means.

Clean Code Easy to Customize

Infinity Slider use the most clear and clean Code that we could create. The HTML’s Structure and the CSS’s Style are really simple and intuitive, it allows you to create and customize your slider without an advanced knowledge of HTML or CSS but simply reading our documentation.

Page & Slider Loader

This Loader allows you to use it for the full-page (or the full-site), only for the slide or leave the page without a loader. We offer you the best way to create your slides and to load it without any kind of problem.

Font Awesome & Site Style

To improve your experience with Infinity Slider, we have included Font Awesome, Bootstrap and Custom Style. This feature will help you to focus on the slider and provide you a set of 350+ retina ready icons, a responsive structure and some few class to develop your slider and site.
